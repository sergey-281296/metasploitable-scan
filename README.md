# Metasploitable Scanning Homework
## DevOps Engineer Training

This repository contains results of scanning Metasploitable2 VM with nmap.

### Files:
- `final_homework_answer.txt` - answers to homework questions
- `services.txt` - service version scan
- `syn_scan.txt`, `fin_scan.txt`, `xmas_scan.txt` - different scan types
- `udp_scan.txt` - UDP scan results
